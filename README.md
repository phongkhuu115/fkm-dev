# Personal Portfolio

## **Create Project**

First to init project, run

```
npx create-react-app <prj-name>
```

After remove some unnecessary folder, add `components` and `assets` folder in `src`. Add some component folder, the folder will now look like this

```
└───src
    ├───assets
    └───components
        ├───body
        ├───contact
        ├───header
        ├───projects
        └───skills
```

## **Package Used**

#### **Tailwind CSS**

CSS library

#### **react-simple-typewriter**

This package is used for type writting effect

#### **gsap**

This package is used for transition in the header

#### **react-icons**

This package is used for icons in the project
